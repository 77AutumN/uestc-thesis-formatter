# 电子科技大学研究生学位论文格式校对清单

> 基于《电子科技大学研究生学位论文撰写规范》提炼，经实战排版验证。

## 封面页
- [ ] 校名"电 子 科 技 大 学"使用正确间距
- [ ] 英文校名 "UNIVERSITY OF ELECTRONIC SCIENCE AND TECHNOLOGY OF CHINA"
- [ ] 校徽清晰、居中
- [ ] 论文题目完整、正确换行
- [ ] 学科专业、学号、作者姓名、指导老师、学院均已填写

## 扉页（学位论文页）
- [ ] 分类号已修改为正确值（非默认 TP309.2）
- [ ] 密级正确（通常为"公开"）
- [ ] UDC 号已填写或留空
- [ ] 无红色占位提示文字（"题名和副题名"、"作者姓名"、"姓名、职称、单位名称"）
- [ ] 申请学位级别正确（硕士/博士）
- [ ] 学科专业正确
- [ ] 提交日期和答辩日期已填写

## 摘要
- [ ] 中文摘要标题为"摘 要"（字间有空格）
- [ ] 中文关键词以"关键词："开头，词间用逗号分隔
- [ ] 英文摘要标题为 "ABSTRACT"
- [ ] 英文关键词以 "Keywords:" 开头

## 目录
- [ ] 自动生成，层级正确（章、节、小节）
- [ ] 页码对齐，无溢出

## 正文格式
- [ ] 章标题：三号黑体，居中
- [ ] 节标题：四号黑体，左对齐
- [ ] 小节标题：小四号黑体，左对齐
- [ ] 正文：小四号宋体
- [ ] 行距：1.5 倍行距（固定值 20pt）
- [ ] 首行缩进：2 个中文字符

## 页眉页码
- [ ] 奇数页页眉：章标题
- [ ] 偶数页页眉：论文题目
- [ ] 页码格式：阿拉伯数字，居中

## 参考文献
- [ ] 标题为"参 考 文 献"
- [ ] 编号格式 [1], [2], ... 连续
- [ ] 正文中有对应的 \cite{} 引用（如果初稿已标注）

## 致谢
- [ ] 标题为"致 谢"
- [ ] 内容完整

## 三线表（如有）
- [ ] 顶线和底线粗（1.5pt）
- [ ] 中间线细（0.75pt）
- [ ] 无竖线
