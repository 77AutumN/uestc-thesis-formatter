# D14 — 外文图强宽度强换行

## 症状
外文译文末尾 p48-49 几乎全是空白, 只显示 1-2 个公式。

## 根因
extractor `_emit_foreign_section` 把所有 drawing 都用 `\noindent\includegraphics[width=0.95\textwidth]{...}\\[1ex]`, 导致小尺寸公式被强拉满整页 + 强换行, LaTeX 不能在 `\\[1ex]` 处 break, 连续图挤满 p48-49 形成大量空白。

## 修复
extractor 用 PIL 探每张图实际宽度, W>600 才加 `width=\textwidth`, 小图 natural size; 去掉 `\noindent` 和 `\\[1ex]`, emit 普通 `\includegraphics{...}` 让 LaTeX 自然流水。

## 注
graphicx 不支持 max-width (vendor cls 没装 adjustbox), 所以走 PIL probe 方案。

## 如何识别
- PDF 视觉: 外文末尾大空白
- misc/foreign_*.tex 含 `\\[1ex]` 或 `width=0.95\textwidth`
