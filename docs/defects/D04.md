# D04 — author 字段用逗号分隔 → bibtex 解析崩

## 症状
bibtex 报错 "Too many commas in name 1"。

## 根因
parse_* 函数直接把逗号分隔字符串写进 `author = {...}`。bibtex 标准要求多作者用 ` and ` 分隔, 逗号是 "lastname, firstname" 反序分隔符。

## 修复
新增 `sanitize_author_list()` 助手, 把 `[,，、;；]` 分隔的作者列表统一转 ` and ` 分隔, 末位 `等`/`et al` → `others`。

## 如何识别
- ref.bib author 字段用 `,` 分多名作者
