# D42 — caption-lookalike 解说段被误 strip

## 症状
CASE-A docx para[243] = "图3-4 给出了不同 fs/B 取值下各阵元 |Δτ_m| 的分布..." 一整段解说正文, 产物 chapter ch03.tex 中 fig 3-4 caption 之后**直接跳到** fig 3-5 caption, 整段解说丢失. 同 case16 还有 fig 3-5 / 3-7 / 4-1 后的 4 处解说段全丢.

## 根因
`recover_figures.inject_into_chapter` (D38 后的代码) 用正则 `r"^\s*图\s*(\d+)\s*[.\-－]\s*(\d+)\s*\S.*$"` 匹配 caption-only 行后, 当 record 为 None 或已放置时**整段 strip 当 caption 占位行处理**. 但 "图X-Y 给出了..." 实际是正文不是 caption, 被误判.

## 修复 (Round 9 W3 shared, 2026-05-05)
拆 `LABEL_RE` (匹配 prefix + tail) + `classify_figure_line(tail) -> "caption_anchor"|"caption_lookalike_body"`.

启发式 (Codex 节 1.2):
- `caption_lookalike_body`: tail > 60 / 含 `给出|展示|表明|说明|可以看出|如图` 且 > 25 / 含句末标点 `。；;！？!?` → **必须保留, 永不 strip**
- `caption_anchor`: 短段, 未触发上述. 与 record 关联则替换为 figure block, 已放置则 strip duplicate

保守原则: 多保留 caption-lookalike 是 P1/P2, 正文丢失是 P0. unmatched 短段也保留 (不无条件 strip). `source_manifest.paragraphs[].caption_role` 字段同步加供 risk-router 用.

## 如何识别
- docx body 段以 "图X-Y" 开头 + 长解说 + 触发关键词
- 产物 chapter 中 figure block 后立即跳 next caption (无解说) → 段被 strip
- product_audit Check 10 v2 高置信 missing_high 命中
