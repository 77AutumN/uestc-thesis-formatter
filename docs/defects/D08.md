# D08 — Word 章节标题无 Heading 样式

## 症状
pandoc AST 看到 `OrderedList` / `Para`, 而非 `Header`。`find_chapters()` 报 0 章节, 流水线断。

## 根因
极简风格论文用纯文本作章节标题, 子节编号定位但 OL 文本不是"第X章 XXX"格式。

## 当前 workaround (CASE-A)
备份原 docx + 用 python-docx 注入 5 个 Heading 1 段 (基于 ToC 推断标题位置)。手工 ~3 分钟, 需用户授权改原稿 (违反 P0 但只改样式不动文字)。

## 长远修复 (TODO Track A2)
ToC → 正文章节反向匹配: 抽取 docx ToC 条目得到章节列表, 扫描正文找首次出现的"X.1" (子节) 位置, 将其前一段判为章节起点。

## 如何识别
- AST `block.t` 分布无 Header
- find_chapters() 报 0 章节
- docx 内 1 个 Heading 1 段都没
