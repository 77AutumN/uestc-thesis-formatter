# D02 — Step 3/3.5 顺序倒置 → vendor 模板覆盖案例 bib

## 症状
编译时 bibtex 报 58 个 cite key undefined, ref.bib 内容是模板默认条目而非案例条目。

## 根因
Step 3 写 case ref.bib, Step 3.5 紧接着复制 vendor 模板 (含 stale ref.bib) 到 template_dir, 把刚生成的 case ref.bib 覆盖。

## 修复
把 Step 3.5 (`step3_5_assemble`) 挪到 Step 3 (`step3_generate_bib`) 之前, 先复制模板再生成 bib。

## 如何识别
- 案例 ref.bib 的条目名跟预期截然不同
- bibtex 全军覆没 (≥80% cite undefined)
