# D12 — IEEE 格式英文文献被丢

## 症状
ref.bib 26 条 < cite_map 32 条 → main.bbl 缺 6 条 → PDF 末尾参考文献不全。

## 根因
refs_to_bib 只识别 GB/T 7714 中文格式, 不带这些标志的英文文献直接 SKIP。

## 修复
SKIP 之前加启发式: 含双引号标题 / IEEE/ACM/SIAM/Springer 关键词 / Conference/Proceedings/Transactions/Journal 字样的行 → `@misc{key, title={...}, note={原始文本}, year={...}}` fallback。adapter 加 `\nocite{*}` 强制让 ref.bib 全部条目进 main.bbl。

## 如何识别
- refs_report.json 大量 SKIP
- PDF 参考文献页编号断裂
