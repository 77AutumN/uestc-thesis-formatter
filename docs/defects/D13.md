# D13 — LibreOffice 公式 PNG 整页空白

## 症状
用户报"论文中所有公式都看不见"。截图显示公式编号 (2.1)/(2.2) 出现, 公式本体位置完全空白。

## 根因
PIL probe 公式 PNG: 全是 816×1056 (US Letter @96dpi), 内容只占 0.1-0.2%。`\includegraphics[height=2em]` 把整张几乎全白画布缩到 24pt 高, 公式被压成约 1 像素。

## 修复
recover_equations.py 加 Pillow auto-crop (W≥600 + 内容占比 < 85% 触发, 加 8px padding 后 crop 到非白 bbox); render_equation_block height 调 2.6em + keepaspectratio。

## 陷阱
trim 逻辑只在 newly-converted 时跑, cached PNG 跳过。需要 standalone helper 一次性扫 media/ 全部 PNG。

## 如何识别
- PIL: PNG 尺寸 ≥600px + 非白 < 1%
- PDF 视觉: 公式编号有, 本体空
