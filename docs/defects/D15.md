# D15 — 外文文字图分离

## 症状
外文译文显示"先 5 页文字段落连续, 再 1.5 页公式连续", 原 docx 是文字与公式交错。

## 根因
v1 `_emit_foreign_section` 写法 — 一次性 dump 文字然后一次性 dump 图, 丢失原始顺序。

## 修复
重写 _emit_foreign_section 直接遍历 docx XML 段 (从 section 标题之后到下一个 section 标题之前), 逐段判别有文字 emit 文字 / 有 drawing emit \includegraphics, 完全保持 paragraph order。

## 如何识别
- PDF 视觉: 译文连续多段文字后接连续多图, 非交错
