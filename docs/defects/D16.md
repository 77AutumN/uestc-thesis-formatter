# D16 — 外文译文段落扁平化层级恢复

## 症状
PDF 外文译文页内部论文标题/作者/摘要/节标题全跟正文同字号, 跟黄金 PDF 视觉差距大。

## 根因
客户原稿译文段全部 pStyle=(none) 扁平化, extractor v1 emit 全部按普通段处理。规范 §2.5 要求"独立元素用对应字体字号"; 数据源缺层级时, 流水线应当能用启发式恢复结构。

## 修复
加 `_classify_foreign_para()` 状态机 + `_latex_decorate()` 装饰函数 (~80 行)。每段计算类型 (paper_title / authors / abstract_marker / section_title / subsection_title / body), 然后按类型 emit:
- paper_title → `\begin{center}\bfseries\zihao{4} ... \end{center}`
- authors → `\begin{center} ... \end{center}`
- abstract_marker → `\noindent\textit{摘要 ——}\ ...`
- section_title (`第X部分`) → `\section*{...}`
- subsection_title (`X.Y XX`) → `\subsection*{...}`
- 其它 → 普通段 (safe fallback)

仅对外文译文做层级化, 外文原文跳过 (几乎全图)。

## 启发式准确率
章标题 100% / 论文标题 100% / 作者行 100% / 摘要标记 100% / 节标题 ~70-80%。

## P0 合规
只装饰样式宏, 不改作者文字。

## 如何识别
- PDF 视觉对比黄金: 译文标题/作者/摘要与正文同样式
- misc/foreign_translation.tex 无 `\begin{center}\bfseries` 标记
