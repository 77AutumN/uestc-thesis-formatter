# D10 — extractor 不识别外文 section

## 症状
PDF 缺 spec §1.1 #20-21 必含 section, 本科级硬错误。

## 根因
`detect_special_sections()` 只识别 acknowledgement/references/accomplishments, 从未为 foreign 段建文件。adapter 也没 emit `\input{misc/foreign_*}` 的代码。

## 修复 (三处协同)
1. extractor 加 foreign_original / foreign_translation 检测, Step 12 写 extracted/foreign_*.txt
2. **关键**: foreign 原文常以英文论文截图嵌入, Step 12 同时走 docx XML 抽该段 `<w:drawing>` rIds → 直接生成 `\noindent\includegraphics[width=0.95\textwidth]{media/imageN.png}` 行
3. adapter 新增 emit_foreign_appendix() — bachelor 才输出
4. run_v2.py Step 3.5 加 3.3b 子步把 extracted/foreign_*.txt → misc/foreign_*.tex

## 已知限制
foreign 原文是图片时只抽 PNG (跳 WMF), 部分 docx 截图存为 WMF 时漏。

## 如何识别
- PDF grep "外文资料原文" = 0
- profile.has_foreign_literature=true 但 main.tex 无 \input
