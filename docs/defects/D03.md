# D03 — cite key 含空格逗号 → bibtex 崩

## 症状
bibtex 报错 "White space in argument", entry key 形如 `muta o ,izumi j ,...2023`。

## 根因
`generate_citekey()` 对英文作者列表只取 `parts[-1].lower()`, split 用 `·`(中点), 所以 parts 就是整串字符串, key 直接吞下空格逗号。

## 修复
新增 `sanitize_citekey()` 助手, 在 `generate_citekey()` 末尾过滤所有非 `[a-zA-Z0-9_]` 字符, 首字符若是数字加 `r` 前缀。

## 如何识别
- ref.bib 任一 KEY 含 ` ` / `,` / 中文
