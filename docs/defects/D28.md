# D28 — 缺 noreminder 致 CLS 红字泄漏

## 症状
CASE-A v8 PDF 第 3 页中文摘要末尾、"关键词：..." 下方出现 3 行红色英文:
```
The length of the Chinese Abstract has exceeded the maximum limit of 1 page(s).
```

## 根因
DissertUESTC.cls 内置 `\boolean{usereminder}` 提示机制, 默认开. 中文摘要超 `\chineseabstractpagelimit` 页 + reminder 开 → 输出红字. 设计意图是给作者本人写论文时看的"超长警告".
`\DeclareOption{noreminder}{...}` 关闭, 但 `template_adapter.py:emit_documentclass` 默认没传 noreminder.

## 修复 (Round 8 shared)
`emit_documentclass` 总输出 `\documentclass[<degree>, <print_mode>, noreminder]{DissertUESTC}`.
我们的产物是给客户的最终交付物, **总关 reminder**, 不存在"作者还在写中"的中间态.

## 如何识别
- product_audit Check 7 扫 "has exceeded the maximum limit" 字样 → ❌
