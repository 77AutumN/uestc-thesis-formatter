# D36 — latexmk max_repeat 不足致引用渲染 [?]

## 症状
CASE-A 首跑 PDF 正文 `\cite{}` 渲染 `[?]`, main.log 提示 "Rerun to get cross-references right". xelatex 需 ≥2 后续 pass 让 natbib 正确渲染编号, 但 latexmkrc `$max_repeat=1` 只跑一次 follow-up.

## 根因
`vendor/DissertationUESTC/latexmkrc` 默认 `$max_repeat = 1`, 不够覆盖 BibTeX → xelatex × 2 + bibtex + xelatex × 2 的 5-pass 闭环.

## 修复 (CASE-A 落地)
改 `$max_repeat = 5`. 不影响 marxism profile (该 profile 不用 BibTeX).

## 如何识别
- 首跑 PDF 含 `[?]` 替代 `\cite{}` 编号
- main.log 含 "Rerun to get cross-references right" 多次
