# 电子科技大学学士学位论文撰写格式规范 (Machine-Readable Edition)

> **Source**: 《电子科技大学毕业设计（论文）工作手册》附件二（Pages 40-57）
> **Purpose**: Automated compliance checking oracle for the `thesis-formatter` pipeline (bachelor mode).
> **Usage**: This document is the authoritative reference for bachelor thesis formatting. Each section contains `[CHECKABLE]` tags indicating rules that can be verified automatically via CLS/TEX inspection or PDF measurement.

---

## 第一章 基本结构及装订顺序

### 1.1 论文组成部分

学士学位论文按以下 **1~22** 的顺序装订：

| # | 部分 | 必需性 | 备注 |
|---|------|--------|------|
| 1 | 封面 | 必需 | 文印中心统一制做 |
| 2 | 毕业设计（论文）任务书 | 必需 | 教务处统一表格 |
| 3 | 毕业设计（论文）开题报告表 | 必需 | |
| 4 | 毕业设计（论文）进度计划表 | 必需 | |
| 5 | 毕业设计（论文）初期检查表 | 必需 | |
| 6 | 毕业设计（论文）中期检查表 | 必需 | |
| 7 | 毕业设计（论文）答辩提问记录表 | 必需 | |
| 8 | 毕业设计（论文）成绩考核表 | 必需 | 一式三份，装订一份 |
| 9 | 毕业设计（论文）学术诚信声明 | 必需 | |
| 10 | 中文摘要 | 必需 | 300-500字，3~5个关键词 |
| 11 | 英文摘要 | 必需 | 另起一页，内容与中文摘要相同 |
| 12 | 目录 | 必需 | 从第一章开始，不含前置部分 |
| 13 | 主要符号表 | 可选 | 中英文对照 |
| 14 | 绪论（第一章） | 必需 | ~1000字 |
| 15 | 论文主体 | 必需 | 理工科≥1.5万字，医科和文科≥1万字 |
| 16 | 结论（最后一章） | 必需 | |
| 17 | 致谢 | 必需 | `[CHECKABLE]` ≤200字 |
| 18 | 参考文献 | 必需 | 列在致谢后 |
| 19 | 附录 | 可选 | |
| 20 | 外文资料原文 | 必需 | 含题目、作者、出处和正文 |
| 21 | 外文资料译文 | 必需 | 含题目、作者和译文（无明确字数下限） |
| 22 | 封底 | 必需 | |

> **`[CHECKABLE]`** Pipeline must generate sections in this exact order. Missing required sections = P0 error.
> **CRITICAL 与研究生规范的关键差异**:
> - 本科生有 **外文资料原文/译文**（#20, #21），研究生没有
> - 本科生 **没有** 扉页、独创性声明、图目录、表目录、缩略词表
> - 本科生 **没有** "攻读学位期间取得的成果"
> - 装订顺序：致谢(17) → 参考文献(18)（与研究生一致）
> - 外文译文**无明确字数下限**（注意：1500-2000字要求属于开题报告表，非外文译文）
> - 致谢字数限制：本科≤200字 vs 研究生≤800字

## 第二章 格式规范

### 2.1 封面

- **`[CHECKABLE]`** 封面由文印中心统一制做
- **`[ADVISORY]`** 题目一般不超过 **25 字**，可加副标题
- **`[CHECKABLE]`** 只能填写 **一名** 指导教师，格式为 `姓名 职称`
- 专业以教育部《普通高等学校本科专业目录（2012年）》为准
- **`[CHECKABLE]`** 封面字体：宋体三号加粗，行间距1.5行
- **`[CHECKABLE]`** 英文论文封面：Times New Roman 三号，行间距1.5 lines

### 2.2 摘要

- **`[CHECKABLE]`** 中文摘要 **300-500 字**
- 内容应包括课题设计意义、完成的主要工作、形成的重要结论
- **`[CHECKABLE]`** 关键词 **3~5 个**，从正文或标题中挑选
- **`[CHECKABLE]`** 英文摘要**另起一页**开始书写，内容与中文摘要相同

### 2.3 目录

- **`[CHECKABLE]`** 目录从**第一章**开始，中英文摘要、主要符号表等前置部分不放入目录
- 目录标题："目 录"（中间空一半角空格）
- **`[CHECKABLE]`** 章标题用**黑体小四**，二、三级标题用**宋体小四**，数字用 Times New Roman
- **`[CHECKABLE]`** 章标题不缩进；一级节标题缩进2个半角空格；二级节标题缩进4个半角空格；三级节标题缩进6个半角空格

### 2.4 标题和层次

- **`[CHECKABLE]`** 正文分章节撰写，**每章应另起一页**
- **`[CHECKABLE]`** 标题不超过一行，不加标点符号
- 标题中尽量不采用英文缩写词

**层次格式：**

| 层次名称 | 示例 | 格式 |
|----------|------|------|
| 章 | 第一章 XX…X | 章序及章名**居中**书写，章序与章名间空1个半角字符 |
| 一级节标题 | 1.1 XX…X | 节序**顶格**书写，与标题名间空1个半角字符 |
| 二级节标题 | 1.1.1 XX…X | 阐述内容另起一段书写 |
| 三级节标题 | 1.1.1.1 XX…X | |

> **`[CHECKABLE]`** 各层次的节序及标题不得置于页面的最后一行（防孤行 Orphan Control）

### 2.5 字体和段落

#### 2.5.1 中文论文字体要求

| 内容 | 字体 | 字号 | 对齐 | 段前 | 段后 | 备注 |
|------|------|------|------|------|------|------|
| **章节标题** | 黑体 | 小三 (15pt) | 居中 | 24磅 | 18磅 | `第一章 XXX` |
| **一级节标题** | 黑体 | 四号 (14pt) | 顶格左对齐 | 18磅 | 6磅 | `4.1 实验装置` |
| **二级节标题** | 黑体 | 四号 (14pt) | 顶格左对齐 | 12磅 | 6磅 | `4.2.2 实验装置` |
| **三级节标题** | 黑体 | 小四 (12pt) | 顶格左对齐 | 12磅 | 6磅 | `1.3.4.1 协商系统` |
| **正文** | 宋体 | 小四 (12pt) | 两端对齐(首行缩进) | 0 | 0 | 行距固定值20磅 |
| **表题与图题** | 宋体 | 五号 (10.5pt) | 居中 | — | — | 详见2.6节 |
| **参考文献** | 宋体 | 五号 (10.5pt) | 两端对齐(悬挂缩进) | 0 | 0 | |
| **页眉** | 宋体 | 五号 (10.5pt) | 居中 | 0 | 0 | |
| **页码** | TNR | 小五 (9pt) | 底端居中 | 0 | 0 | |

> **`[CHECKABLE]`** 论文中所用中文字体（除各级标题外）为**宋体**，各级标题用**黑体**
> **`[CHECKABLE]`** 数字和英文统一用 **Times New Roman**
> **`[CHECKABLE]`** 行间距固定值 **20 磅**
> **`[CHECKABLE]`** 所有段落**首行空4个半角字符**起书写
> **`[CHECKABLE]`** 中英文混排标点规则：所有标点符号一律使用**中文输入状态**下的标点符号，但小数点采用英文状态下的样式 `.`

#### 2.5.2 英文论文字体要求

| 内容 | 字体 | 大小 |
|------|------|------|
| 章节标题 | TNR | 15磅加粗 |
| 一级节标题 | TNR | 14磅加粗 |
| 二级节标题 | TNR | 14磅加粗 |
| 三级节标题 | TNR | 12磅加粗 |
| 正文 | TNR | 12磅 |
| 表题与图题 | TNR | 10.5磅 |
| 参考文献及页眉 | TNR | 10.5磅 |

#### 2.5.3 段落补充规则

1. **`[CHECKABLE]`** 若两个标题之间没有文字，第二个标题的段前距设置为 **0 磅**
2. **`[CHECKABLE]`** 目录及参考文献行间距取固定值 **20磅**
3. 不要在一篇参考文献中间换页

**图/表/公式与正文间距：**

| 元素 | 段前 | 段后 |
|------|------|------|
| 图题 | 6磅 | 12磅 |
| 表题 | 12磅 | 6磅 |
| 公式 | 6磅 | 6磅 |

### 2.6 图、表和公式

#### 通用规则
- **`[CHECKABLE]`** 一律采用阿拉伯数字**分章连续编号**：图 2-5，表 3-2，公式（5-1）
- **`[CHECKABLE]`** 图中中文用宋体五号字，英文和数字用 Times New Roman 10.5磅
- 若图或表中有附注，采用英文大写字母顺序编号，附注写在图或表的下方

#### 图 (Figures)
- **`[CHECKABLE]`** 图题**居中置于图的下方**
- 分图号用 (a)、(b) 等表示
- **`[CHECKABLE]`** 有分图时，各分图题按序置于主图题后方，分图题之间用**分号"；"**隔开
- **`[CHECKABLE]`** 图题不能跨页，图与图题不得拆开
- 引用文献中的图，图题右上角标注参考文献序号
- 图题超过一行时，左右缩进2个字，回行后与主图名左对齐

#### 表 (Tables)
- **`[CHECKABLE]`** 表题**居中置于表的上方**
- **`[CHECKABLE]`** 表格线统一用**单线条**，磅值为 **0.5磅**
- 表中中文用宋体五号，英文/数字用 Times New Roman 10.5磅
- 表题超过一行时，表题仍然居中置于表上方，但表名**左对齐**编排
- 全表如用同一单位，则将单位符号移至表头右上角，加圆括号
- 一般不能拆开两页编排
- 引用文献中的表，表题右上角标注参考文献序号

> **⚠️ 与研究生规范的关键差异**：
> - 本科生表格线统一 **0.5磅** 单线条
> - 研究生三线表：上下线 **1.5磅**，内线 **0.75磅**

#### 公式 (Equations)
- **`[CHECKABLE]`** 另起行，**居中**书写
- **`[CHECKABLE]`** 公式序号**右端对齐**，Times New Roman 12磅
- 公式较长时在 `=` 处转行

### 2.7 量和单位

- **`[CHECKABLE]`** 量的符号、常量和变量符号必须采用**斜体**
- **`[CHECKABLE]`** 计量单位符号一律采用**正体**书写
- 计量单位名称可采用国际通用符号或中文名称，但全文应统一，不要两种混用
- 除用人名命名的单位第一个字母大写外，一律用小写字母

### 2.8 引用文献标注

- **`[CHECKABLE]`** 采用**顺序编码制**
- **`[CHECKABLE]`** 置于所引内容最后一个字的**右上角**，阿拉伯数字置于方括号 `[ ]` 中，**12磅字体的上角标**
- 引用单篇：`离散卡尔曼滤波[1]`
- 同一处引用多篇：`[1,5,14-17]`
- 文献作为句子成分时：12磅字与正文排齐，如 `由文献[8,10-13]可知`
- **`[CHECKABLE]`** 不得将引用文献标示置于各级标题处

### 2.9 参考文献格式

- **`[CHECKABLE]`** 按引用顺序附于文末
- **`[CHECKABLE]`** 作者写到第三位，余者写 `, 等` 或 `, et al.`
- 英文作者：名在前（缩写），姓在后（全拼，首字母大写）

**文献类型标志：**

| 类型 | 标志 | 类型 | 标志 |
|------|------|------|------|
| 期刊文章 | [J] | 专利 | [P] |
| 会议论文 | [C] | 标准 | [S] |
| 专著 | [M] | 数据库 | [DB] |
| 学位论文 | [D] | 计算机程序 | [CP] |
| 报纸文章 | [N] | 电子公告 | [EB] |
| 报告 | [R] | 互联网 | [OL] |

**主要格式：**

| 文献类型 | 书写格式 |
|---------|---------|
| 期刊 | `[序号] 作者.文题[J]. 刊名，年，卷号（期号）：起-止页码` |
| 会议 | `[序号] 作者.文题[C]. 会议论文集名 会议地点，会议时间，起-止页码` |
| 专著 | `[序号] 作者.书名[M]. (译者).出版地：出版者，出版年，起-止页码` |
| 学位论文 | `[序号] 作者.文题[D]. 授予单位所在地：授予单位，授予年，起-止页码` |
| 电子文献 | `[序号] 作者.文题[文献类型/载体标志]. 出版地或获得地址：出版者，日期` |

### 2.10 定理环境和证明环境

- **`[CHECKABLE]`** `定理 X.X`、`引理 X.X`、`证明` 等字体为**黑体**，字号为**小四**
- 段前空 **4个半角字符**
- **`[CHECKABLE]`** 证毕符号：黑色方块 **■**，置于证明内容最后一行末尾

### 2.11 脚注

- **`[CHECKABLE]`** 小五号字，两端对齐，单倍行距，段前段后均空 0 磅
- **`[CHECKABLE]`** 序号按页编排，不同页不需要连续
- **`[CHECKABLE]`** 序号采用 `①，……，⑩` 样式
- 中文用宋体小五号，英文和数字用 Times New Roman 9磅
- **`[CHECKABLE]`** 悬挂缩进1.5字符，序号与内容间空1个半角字符

## 第三章 印制要求

### 3.1 页眉

| 部分 | 页眉内容 |
|------|----------|
| 任务书、开题报告表、各种检查表等 | **不设置页眉** |
| 中文摘要、英文摘要、目录 | 分别用各部分内容的标题 |
| 从第一章开始（奇数页） | **"本章标题"** |
| 从第一章开始（偶数页） | **"电子科技大学学士学位论文"** |

- **`[CHECKABLE]`** 页眉字体：中文宋体五号居中，英文/数字 Times New Roman 10.5磅居中
- **`[CHECKABLE]`** 页眉线：**单横线**

> **⚠️ 与研究生规范差异**：偶数页页眉为"电子科技大学**学士**学位论文"

### 3.2 页码

| 部分 | 页码规则 |
|------|----------|
| 任务书、检查表等 | **不设置页码** |
| 中文摘要、英文摘要、目录等前置部分 | **罗马数字**连续编排 |
| 从绪论（第一章）开始 | **阿拉伯数字**连续编排，第一章以奇数页（第"1"页）开始 |

- **`[CHECKABLE]`** 前置部分每部分采用**双面印制**，若某部分页数为奇数，最后一页单面印制（背面空白）
- **`[CHECKABLE]`** 第一章以后连续编排，其他章**不是**一定以奇数页开始
- **`[CHECKABLE]`** 页码位于**页面底端**，**居中**书写；字体 Times New Roman，字号**小五**

> **⚠️ 与研究生规范差异**：本科生正文中其他章节不强制奇数页开始（连续编排）

### 3.3 页面设置

| 纸张规格 | 页边距（左、右） | 页边距（上、下） | 页眉距边界 | 页脚距边界 |
|---------|---------------|---------------|-----------|-----------|
| **A4 (210×297mm)** | **30mm** | **30mm** | **20mm** | **20mm** |

> **`[CHECKABLE]`** 与研究生规范一致

### 3.4 印刷规格

- **`[CHECKABLE]`** 论文正文部分要求**双面印刷**
- 正文前的表格部分除成绩考核表外，均**单面印刷**

## 附录B：本科 vs 研究生规范差异速查表
